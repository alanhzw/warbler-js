# 数组

## 删除数组中的重复元素



### 语法

```js
arrWithoutDupli(arr)
```

### 参数

- `arr`(**Array**) 要去重的数组。

### 返回值

**Array** : 返回一个去重后的新数组。

### 例子


```js
import arrWithoutDupli from 'warblerJS'

const fruits = ["apple", "mango", "orange", "apple", "pineapple", "pineapple", "peach", "mango"]

const result = arrWithoutDupli(fruits);

console.log(result) //=> ['apple', 'mango', 'orange', 'pineapple', 'peach']
```
